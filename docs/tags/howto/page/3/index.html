Comment backuper une arborescence de fichiers et dossiers par FTP ?

Quand on a besoin de faire un backup d’une arborescence de fichiers et qu’on a que FTP comme service, il est possible d’utiliser la commande suivante : ncftpput -u USER -p PASSWORD -R SERVER LOCAL REMOTE : USER : l’identifiant PASSWORD : le mot de passe SERVER : l’IP ou nom du serveur FTP LOCAL : le dossier source REMOTE : le dossier destinataire L’option -R permet de faire le transfert en récursif, c’est à dire que les dossiers sont automatiquement créés, et c’est l’option killer par rapport au client FTP standard. »
