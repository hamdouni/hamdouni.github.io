Speech Tasks : a voice driven todo webapp

With the Web Speech Api, some good javascript libs and several lines of code, you can easily add speech recognition in your web application. All is done thanks to AnnYang JS lib made by Tal Ater, plumbing with Angular Light by Oleg Nechaev and served by a Go http server. (By the way, I love opensource community !) Try the demo ! See the code ! And thanks to Mark McGranaghan for his tutorial to deploy go app on Heroku »
