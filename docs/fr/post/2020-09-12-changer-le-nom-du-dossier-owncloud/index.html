Changer le nom du dossier ownCloud

samedi 12 septembre 2020 · 1 minute · 134 mots

Pour changer le nom de son dossier de synchronisation ownCloud, on peut utiliser cette astuce sous Linux (probablement aussi sur les autres plateformes).

Tout d’abord, fermez l’application ownCloud en cliquant sur son icône avec le bouton droit puis “Quitter ownCloud”. Cela évitera d’éventuel conflit lors du changement du nom.

Vous pouvez désormais modifier le nom du dossier. Par exemple :

mv ~/owncloud ~/docs

Ouvrez votre fichier de configuration ownCloud : son emplacement dépend de votre plateforme (cf la documentation d’ownCloud pour en savoir plus). Sur mon Linux, j’ai lancé la commande suivante :

vi ~/.local/share/data/ownCloud/owncloud.cfg

Ensuite, repérez la ligne contenant le mot “localPath”. Cela ressemble à :

0\Folders\1\localPath=/home/user/owncloud/

Modifiez le chemin d’accès vers votre nouveau dossier. Par exemple :

0\Folders\1\localPath=/home/user/docs/

Redémarrez l’application ownCloud et vous devriez constater que le dossier est déjà synchronisé !
