PDF.JS : du PDF dans vos pages web sans plugins

lundi 7 mai 2012 · 1 minute · 146 mots

Il y a un an, je faisais une tirade sur le manque d’innovation “web” dans les grosses structures et je prenais pour exemple un cas de visualisation de PDF au sein d’un site.

Je proposais alors l’utilisation de Google Docs qui mettait à disposition une application web de visualisation PDF en JavaScript mais cela ne pouvait marcher qu’avec des PDF publiques (visibles par tous).

Et bien je suis tombé sur le projet PDF.js qui fait exactement la même chose, mais sous forme de librairie JavaScript utilisable partout et donc potentiellement avec des documents privés (intranet, sites restreints, …).

Entièrement faite en JS + HTML5 (utilisation des canevas), cette librairie fait partie des projets du lab de Mozilla, l’éditeur de Firefox. D’ailleurs, il existe une extension firefox qui permet donc de visualiser les PDFs directement dans son navigateur FF sans utiliser de reader (comme chrome de google)
