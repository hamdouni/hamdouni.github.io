Police de caractère dans la console

mardi 29 mars 2022 · 1 minute · 62 mots

Avec des écrans 4K, la console linux devient illisible : la police de caractère est trop petite. Alors comment faire pour agrandir sa taille pour avoir une console lisible après démarrage ?

Il suffit de modifier le fichier de configuration par défaut de la console pour y indiquer une plus grande police :

> cat /etc/default/console-setup
ACTIVE_CONSOLES="/dev/tty[1-6]"
CHARMAP="UTF-8"
CODESET="guess"
FONTFACE="Terminus"
FONTSIZE="16x32"
FONT='Lat7-Terminus32x16'
