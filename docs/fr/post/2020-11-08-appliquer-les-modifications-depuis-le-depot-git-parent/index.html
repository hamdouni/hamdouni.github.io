Appliquer les modifications depuis le dépôt git parent

dimanche 8 novembre 2020 · 1 minute · 111 mots

Cette note explique comment appliquer les nouvelles modifications depuis le dépôt git parent sur votre dépôt hérité (fork).

Si ce n’est pas déjà fait, ajoutez un “remote” vers le dépôt parent comme suit :

git remote add parent https://github.com/project/name.git

Ensuite, récupérez toutes les modifications depuis la branche master (ou toute autre branche qui vous convient) depuis le dépôt parent :

git fetch parent master

Maintenant, appliquez les modifications sur votre branche :

git rebase parent/master

Il se peut que vous ayez des conflits si vous avez modifié votre dépôt. Vous devez régler tous les conflits avant de continuer.

Finalement, poussez (en forçant) toutes vos modifications :

git push --force origin master
