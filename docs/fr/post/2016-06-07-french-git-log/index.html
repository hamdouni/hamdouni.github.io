French Git Log

mardi 7 juin 2016 · 1 minute · 108 mots 

git log --pretty='format:%Cblue%h%Creset %ad %Cred%<(8,trunc)%an%Creset%Cgreen%d%Creset %s' --date=short

Utiliser un “pretty format” pour afficher l’historique de Git avec des couleurs et des columns à taille fixe :

%Cblue … %Creset
indique le début et la fin d’une portion colorée. Le rouge (red) et le vert (green) sont aussi utilisables.
%h
format court du hash du commit
%ad
format court de la date du commit avec l’option complémentaire - date=short
%<(8,trunc)
fixe la taille du prochain élément à 8 caractères (complété d’espace à droite si nécessaire ou coupé si trop long)
%an
le nom de l’auteur du commit
%d
nom de la référence (branch, tag, …)
%s
commentaire du commit
