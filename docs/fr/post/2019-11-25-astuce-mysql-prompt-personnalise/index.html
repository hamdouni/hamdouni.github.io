Astuce Mysql : prompt personnalisé

lundi 25 novembre 2019 · 1 minute · 96 mots

On peut facilement modifier le prompt du client mysql, et ainsi repérer dans quelle instance MySQL on se trouve.
C’est très utile lorsqu’on doit jongler entre plusieurs bases, et qu’on finit par ne plus savoir dans laquelle on se trouve :-)

Par exemple, si je travaille sur le projet FooBar, je peux utiliser le client comme suit :

mysql --prompt="@FooBar> "

J’obtiens dans ma console, quelque chose comme ça :

Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

@FooBar> show databases;

Et voilà, plus jamais perdu dans mes bases :-D
