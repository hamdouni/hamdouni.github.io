GIT – stop tracking files

If you have some files that you need to have in your repo but don’t need to have updates, git allows it : git update-index --skip-worktree <FILES> Then a “git status” won’t show those files until you do : git update-index --no-skip-worktree <FILES> To show the skipped files : git ls-files -v . | grep ^S –skip-worktree is a better solution than –assume-unchanged because status will not get lost once an upstream change is pulled. »
